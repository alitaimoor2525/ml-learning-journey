# Types of Variables in Machine Learning

## Introduction
In Machine Learning and Statistics, variables represent different types of data. Understanding variable types is very important for data preprocessing, visualization, and model selection.

---

## 1. Numerical Variables (Quantitative Data)

### Definition
Numerical variables are variables that contain numbers and can be measured.

### Types of Numerical Variables

#### a) Discrete Variables
- Countable values (usually integers)
- Cannot take decimal values

**Examples:**
- Number of students in a class
- Number of cars
- Number of calls received

#### b) Continuous Variables
- Can take any value within a range
- Includes decimal values

**Examples:**
- Height
- Weight
- Temperature
- Salary

---

## 2. Categorical Variables (Qualitative Data)

### Definition
Categorical variables represent categories or labels instead of numbers.

### Types of Categorical Variables

#### a) Nominal Variables
- No specific order
- Categories are just labels

**Examples:**
- Gender (Male/Female)
- Color (Red, Blue, Green)
- City names

#### b) Ordinal Variables
- Have a meaningful order or ranking

**Examples:**
- Education level (School, College, University)
- Satisfaction (Low, Medium, High)
- Grades (A, B, C)

---

## Summary Table

| Type | Subtype | Description | Example |
|------|--------|------------|--------|
| Numerical | Discrete | Countable numbers | No. of students |
| Numerical | Continuous | Measurable values | Height, Weight |
| Categorical | Nominal | No order | Gender, Color |
| Categorical | Ordinal | Ordered categories | Grades, Levels |

---

## Key Differences

- Numerical = Numbers
- Categorical = Labels
- Discrete = Countable
- Continuous = Measurable
- Nominal = No order
- Ordinal = Has order
